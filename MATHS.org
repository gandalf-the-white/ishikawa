Function transpose_matrix(M):
    m = number of rows in M
    n = number of columns in M
    T = empty matrix

    For j from 0 to n-1:
        Initialize row as an empty list
        For i from 0 to m-1:
            Append M[i][j] to row
        Append row to T

    Return T
